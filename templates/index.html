Configuration Documentation

Required: You must place a secrets.json file in the root folder containing your OPENAI_API_KEY, ELEVEN_API_KEY, or GEMINI_API_KEY before running the server.
1. persona.json (Characters & Overrides)
Defines the AI characters. You can override global model routing and interruption behavior on a per-persona basis. 
{
  "personas": [
    {
      "name": "Angry Chef",
      "persona": "You are a highly aggressive chef running a busy kitchen.",
      "llm_provider": "openai",
      "tts_provider": "elevenlabs",
      "voice_id": "JBFqnCBsd6RMkjVDRZzb",
      "interruption_config": {
        "MODE": "probabilistic",
        "STRATEGY_MATRIX": {
          "COMPETITIVE": {"RESUME": 100, "BRIDGE": 0, "YIELD": 0, "EXIT": 0}
        }
      }
    }
  ]
}
2. session_config.json (Flow & Surveys)
Controls how long the experiment lasts and what questions are asked during the post-session survey. Supports two modes:

Individual Mode (default): One survey after each persona session. 
{
  "MAX_TURNS": 5,
  "SURVEY_TYPE": "individual",
  "SURVEY_QUESTIONS": [
    { "question": "Did the bot sound angry?", "expected_type": "options", "allowed_options": ["yes", "no"] }
  ]
}
Comparative Mode: Groups personas into sets of SET_SIZE. One comparative survey appears after the full set is completed. Use SET_LABELS to name each condition. 
{
  "MAX_TURNS": 5,
  "SURVEY_TYPE": "comparative",
  "SET_SIZE": 3,
  "SET_LABELS": ["Style A", "Style B", "Style C"],
  "SURVEY_QUESTIONS": [
    {
      "question": "The reaction felt natural.",
      "expected_type": "matrix_likert",
      "allowed_options": ["1","2","3","4","5"]
    },
    {
      "question": "Which style was best for this character?",
      "expected_type": "forced_choice",
      "allowed_options": []
    },
    {
      "question": "Explain your choice.",
      "expected_type": "text",
      "allowed_options": []
    }
  ]
}
Question TypeModeDescription
matrix_likertComparativeRate each style side-by-side on a Likert scale. Add "reverse_coded": true for reverse-scored items.
forced_choiceComparativePick one of the SET_LABELS as the best.
numberBothNumeric input, validated against allowed_options.
optionsBothSelect from a predefined list.
textBothFree-form text response.
3. model_config.json (Global Providers)
Sets the fallback models and voices if a persona does not specify them explicitly. 
{
  "LLM_PROVIDER": "gemini",
  "GEMINI_MODEL": "gemini-2.5-flash",
  "TTS_PROVIDER": "openai",
  "OPENAI_TTS_MODEL": "gpt-4o-mini-tts",
  "OPENAI_DEFAULT_VOICE": "coral"
}
4. interruption_config.json (Logic Matrix)
The global probability matrix defining how the bot reacts to being interrupted. 
{
  "MODE": "probabilistic",
  "INTENTS": {
    "COMPETITIVE": "User disagrees, corrects, or tells you to be quiet."
  },
  "STRATEGY_MATRIX": {
    "COMPETITIVE": {"RESUME": 80, "BRIDGE": 20, "YIELD": 0, "EXIT": 0}
  }
}
5. secrets.json (API Keys)
Contains the API keys required to generate responses and voice. Because this is a public platform, do not share this file with anyone. Uploading it stores it securely in the database linked only to your researcher account. 
{
  "OPENAI_API_KEY": "sk-proj-...",
  "ELEVEN_API_KEY": "sk_...",
  "GEMINI_API_KEY": "AIza..."
}
